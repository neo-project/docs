命名空间：[Neo.SmartContract.Framework.Services.Neo](../neo.md)

## 语法

```c#
public class StorageMap
```

## 属性

| 名称                                       | 说明         |
| ---------------------------------------- | ---------- |
| Context | 存储上下文 |
| Prefix | 前缀 |

## 构造方法
